# 美术资产占位符

此目录用于存放占位符资产，正式美术资产生产前使用。

## 命名规范

- 卡牌插图：`card_[id].png`（2:3 比例，建议 400×600px）
- 敌人立绘：`enemy_[id].png`（建议 512×768px）
- 场景背景：`bg_[scene].png`（1920×1080px）
- UI 图标：`icon_[name].png`（128×128px）

## Midjourney Prompt 模板

所有牌卡美术必须携带的基础风格标签：

```
Chinese jianzhi paper-cut art style, ink wash painting aesthetic, 
black vermillion gold color palette, flat 2D folklore illustration, 
high contrast silhouette, traditional Chinese supernatural theme, 
card game art --ar 2:3 --style raw --q 2
```
